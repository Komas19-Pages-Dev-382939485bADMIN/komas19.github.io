Dearus Removal

Lets start by fixing the boot configuration data file missing required information.

You will need:
- An .iso file of windows/a usb drive containing an windows setup (we have an compact one here)
- If you don't have an usb drive containing an windows setup, get a usb drive (must be around 3-8GB minimum)
- You will also need an working computer for burning the iso file into the usb drive.

Note: If you are just removing it from a virtual machine, you just need the .iso file, you also don't need to do the next step about rufus

Ok so, install rufus from here, then burn the iso file into your usb drive.

Now boot into the windows setup, then do Shift + F10 (Fn + Shift + F10 if your computer contains the Fn key on the keyboard)

Do:
bootrec /fixmbr
bootrec /fixboot
bootrec /rebuildbcd

Now, it should identify the Windows Installation, do on the command prompt:
wpeutil reboot

Now it should reboot, you can now remove the usb drive from the computer.

The computer will now boot successfully, but we are not done!

NOTE: IF YOU WANT TO AUTOMATE THE PROCESS FOR FIXING REGEDIT WHILE DOING THE CHANGES FOR YOU: Download this

Download this file: fixregedit.vbs

Put it in the system that is infected with Dearus, launch the .vbs then

Do Windows + R then type:
regedit

now, go to HKEY_CURRENT_USER -> Software -> Policies -> Microsoft -> Windows -> System, in the system key, you should see "DisableCMD" on the right of the window, double click it then replace 1 by 0

and go to HKEY_CURRENT_USER -> Software -> Microsoft -> Windows -> CurrentVersion -> Policies -> System, in the system key, you should see "DisableTaskMgr" on the right of the window, double click it then replace 1 by 0

and go to HKEY_CURRENT_USER -> Policies -> Microsoft -> MMC -> {8FC0B734-A0E1-11D1-A7D3-0000F87571E3}, you should see "Restrict_Run" on the right of the window, double click it then replace 1 by 0

You are done now! No more changes are required as these are the only changes being done!
