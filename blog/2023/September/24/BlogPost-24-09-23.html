About the outage September 24th of 2023

Well it has been a long time, again no?
The website has been down for a few months, we have fixed it, lets talk about the outage now
  1. What Happened? A few months ago, the website went fully offline, with no domains working
  2. Why did this happend? https://komas19.pages.dev used to redirect to https://komas19.cf and that domain, expired, and the provider we used, is gone for some time
  3. How was this fixed? We simply removed the way that it redirects to https://komas19.cf
  4. What would you recommend to other people owning a site? First off, check regularly, every week for example, if the website is working. Secondly, do not make it redirect to a custom domain, keep the old domain just in case the issue we had happens to you, and for any pages, make it get the styles.css for example, by the old domain

And thats all!
See you next time!
